Tutorial criação de projeto Django




* Passo 1 : Criar um Novo Projeto Django


Você pode criar um novo projeto Django usando o comando `django-admin` ou `manage.py`. Vamos usar `manage.py` para criar um projeto chamado "meuprojeto":




django-admin startproject meuprojeto




Este comando criará um diretório chamado "meuprojeto" com a estrutura inicial do projeto.


* Passo 2: Configurar o Banco de Dados


Abra o arquivo `settings.py` dentro do diretório "meuprojeto" e vá até a seção `DATABASES` para configurar as informações de conexão com o banco de dados.




DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.sqlite3',  # Use SQLite por padrão
        'NAME': os.path.join(BASE_DIR, 'db.sqlite3'),  # Caminho para o arquivo do banco de dados
    }
}




* Passo 3: Criar as Tabelas do Banco de Dados


Agora, crie as tabelas do banco de dados definidas em seu projeto:




cd meuprojeto
python3 manage.py migrate




Isso aplicará as migrações iniciais que vêm com o Django.


* Passo 4: Criar um Aplicativo


Os projetos Django podem ser compostos por vários aplicativos. Um aplicativo é uma parte específica de funcionalidade em seu projeto. Vamos criar um aplicativo chamado "meuapp":




python3 manage.py startapp meuapp




* Passo 5: Definir Modelos


Em seu aplicativo "meuapp", você definirá modelos para representar os dados do seu sistema. Os modelos definem a estrutura das tabelas do banco de dados. Edite o arquivo `models.py` dentro do diretório "meuapp" para definir seus modelos.


* Passo 6: Criar Migrações e Aplicar ao Banco de Dados


Depois de definir seus modelos, crie migrações e aplique-as ao banco de dados para criar as tabelas correspondentes:




python3 manage.py makemigrations meuapp
python3 manage.py migrate




* Passo 7: Criar Views e Templates


Crie views para lidar com a lógica de negócios do seu aplicativo e templates para renderizar as páginas HTML. Configure as URLs em `urls.py` para mapear as views às URLs corretas.


* Passo 8: Iniciar o Servidor de Desenvolvimento


Para testar seu projeto localmente, execute o seguinte comando para iniciar o servidor de desenvolvimento do Django:




python3 manage.py runserver




Acesse seu projeto em `http://localhost:8000/` no seu navegador.
